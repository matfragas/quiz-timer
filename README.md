Quiz Timer ⏱️

Un widget interactif pour gérer un quiz de 80 questions en 60 minutes.
Fonctionnalités

✅ Timer circulaire - Visualisation du temps restant
✅ Suivi des questions - Entrez le numéro de la question en cours
✅ Calcul du rythme - Vérification si vous êtes en retard ou en avance
✅ Statistiques en temps réel - Temps par question, questions/minute
✅ Alertes intelligentes - Avertissements si vous dépassez le rythme idéal
✅ Sauvegarde locale - Vos progrès et notes sont sauvegardés automatiquement
✅ Notes rapides - Prenez des notes pendant le quiz
Utilisation

    Ouvrez index.html dans votre navigateur
    Cliquez sur Démarrer pour commencer le timer
    Entrez le numéro de la question actuelle au fur et à mesure
    Consultez les statistiques pour vérifier votre rythme
    Cliquez sur Pause pour faire une pause
    Cliquez sur Réinitialiser pour recommencer

Temps idéal

    Par question : ~45 secondes
    Rythme cible : 1.3 questions/minute
    Total : 60 minutes pour 80 questions

Stockage

Vos données sont automatiquement sauvegardées dans le stockage local du navigateur (localStorage).
Technologie

    HTML5
    CSS3 (Flexbox, Grid, Animations)
    JavaScript vanilla (pas de dépendances)
