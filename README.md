# Quiz Timer ⏱️

Un timer simple et efficace pour gérer un quiz de 80 questions en 60 minutes.

## Fonctionnalités

✅ **Décompteur global** - Affichage du temps restant total  
✅ **Numéro de question** - Suivi de la question actuelle (1-80)  
✅ **Timer par question** - 45 secondes par défaut avec changement de couleur:
   - 🟢 Vert: Vous êtes dans les temps
   - 🟠 Orange: À 80% du temps utilisé
   - 🔴 Rouge: À 90% du temps utilisé

✅ **Bouton "Question suivante"** - Réinitialise le timer de la question  
✅ **Alertes intelligentes** - Vous averti si vous êtes en retard/avance  
✅ **Sauvegarde locale** - Votre progression est sauvegardée automatiquement  

## Utilisation

1. Ouvrez `index.html` dans votre navigateur
2. Cliquez sur **Démarrer** pour commencer le quiz
3. Le timer global compte les 60 minutes restantes
4. Le timer de question montre le temps pour la question en cours (45 sec)
5. Quand vous finissez une question, cliquez sur **Question suivante**
6. Consultez les avertissements pour vérifier votre rythme

## Temps idéal

- **Par question**: 45 secondes
- **Total**: 60 minutes pour 80 questions
